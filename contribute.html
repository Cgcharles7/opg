Contributing Conjectures

You can add new conjectures by creating a JSON file in the conjectures/ folder and adding its filename to manifest.json. Each conjecture should follow this format: 

{
  "uuid": "unique-id",
  "name": "Riemann Hypothesis",
  "link": "https://en.wikipedia.org/wiki/Riemann_hypothesis",
  "summary": "States that all nontrivial zeros of the zeta function ...",
  "authors": ["Bernhard Riemann"],
  "kwds": ["Analysis", "Number Theory"],
  "type": "conjecture"
}

The order of keywords determines the hierarchical placement: e.g. ["Algebra", "Group Theory"] appears under Algebra → Group Theory.

After adding your JSON file, update manifest.json with its filename: 

[
  "riemann_hypothesis.json",
  "twin_prime_conjecture.json"
]

Then open index.html in your browser — your conjecture should appear.
